# Delete previously saved rounds from the round history screen

## Concept

Delete the scorecard for a previously saved round by sliding the row left based on the common iphone implementation pattern

## Key Decisions

- **Navigation method:** Swipe left on the row to delete, or enable the delete button
- **Persistence:** Delete based on round id
- **Navigation:** None

## User Stories

- As a golfer, I want to be able to delete a previously saved round when viewing the 'round history'.

## Database Schema Considerations

- Existing schema should support deleting previously saved rounds based on round id.

## UI Placement

- Lives within the **On Course** tab.
